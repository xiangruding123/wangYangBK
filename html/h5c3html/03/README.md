animation.html文件
CSS3 animation 属性：

使用简写属性，将动画与 div 元素绑定

div
{
animation:mymove 5s infinite;
-webkit-animation:mymove 5s infinite; /* Safari 和 Chrome */
}

将 div 元素中的文本分为三列！！
div
{
-moz-column-count:3; /* Firefox */
-webkit-column-count:3; /* Safari 和 Chrome */
column-count:3;
